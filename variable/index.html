    public appLogger: Provider source 

    import {appLogger} from 'wootils/wootils/node/logger.js'

    The service provider that once registered on the app container will set an instance of Logger as the appLogger service. The difference with the regular logger is that this one uses the packageInfo service in order to retrieve the name of the project and use it as messages prefix.

    Example:

    // Register it on the container
container.register(appLogger);
// Getting access to the service instance
const appLogger = container.get('appLogger');
